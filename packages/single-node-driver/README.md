## Pre-requisites
-`python 3.6+`
-`pip3`
- `node` version 16.0.0 or higher
- Make sure `cwltool` is installed and available in the path

```bash
echo "export PATH=\"`python3 -m site --user-base`/bin:\$PATH\"" >> ~/.bashrc
source ~/.bashrc
```

## Installation
1. `npm install`

## Running
2. `npm run dev`
